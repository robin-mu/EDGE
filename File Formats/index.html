Two Tribes File Formats

Two Tribes Engine

  • misc - Generic Structures
  • ean - 3D Animation
  • ema - Material
  • eso - 3D Mesh
  • etx - Texture
  • glyph - Bitmap Font
  • loc - Multilanguage Strings

EDGE

  • level.bin - Level Format
  • cos.bin - Precalculated Cosine Wave
  • font.bin - Font

Toki Tori GBC

  • rem - Level file
  • sav - Save file

Toki Tori 2

  • ma - File Archive
  • sprt - Spritestrip Metadata
  • trigger - Particle Emitter

All multi-byte primitive types (int, float, etc.) are assumed to be in little-endian byte order except where specified otherwise.

To avoid confusion, primitive types are defined as follows:

  • char - 8-bit unsigned integer
  • short - 16-bit signed integer
  • int - 32-bit signed integer
  • uXX - XX-bit unsigned integer
  • float - 32-bit IEEE 754 floating point
